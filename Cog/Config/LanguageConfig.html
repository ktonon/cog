- (Object) activate_language(key, opt = {}) { ... }

Activate a given language within the scope of the provided block. Either provide key or :ext but not both. If the extension does not match any of the supported languages, the #active_language will not change, but the block will still be called.

Parameters:

  • key (:String) —

    the lanuage identifier. Type cog language list to see the possible values

  • opt (Hash) (defaults to: {}) —

    a customizable set of options

Options Hash (opt):

  • :ext (:String) — default: nil —

    a file extension which will map to a language identifier. Type cog language map to see mapped extensions

Yields:

  • within this block the #active_language will be set to the desired value

Returns:

  • (Object) —

    the value returned by the block

def activate_language(key, opt={}, &block)
  throw :ActivateLanguageRequiresABlock if block.nil?
  opt, key = key, nil if key.is_a? Hash
  key = if opt[:ext]
    ext = opt[:ext].to_s.downcase
    ext = ext.slice(1..-1) if ext.start_with?('.')
    @language_extension_map[ext] unless ext.empty?
  else
    key
  end
  if key
    @active_languages << @language[key]
    r = block.call
    @active_languages.pop
    r
  else
    block.call
  end
end

def language_for(path)
  ext = File.extname(path.to_s)
  ext = path.to_s if ext.empty?
  ext = ext.downcase
  ext = ext.slice(1..-1) if ext.start_with? '.'
  key = @language_extension_map[ext]
  @language[key] if key
end
