Overview

In your projectâ€™s Cogfile, self has been set to an instance of this class.

Typing cog init will create a Cogfile in the present working directory. Cogfile files are used to configure an instance of Cog::Config.

Examples:

project_generators_path 'cog/generators'
project_templates_path 'cog/templates'
project_source_path 'src'
language 'c++'

def project_generators_path(path, absolute=false)
  @config.instance_eval do
    @project_generators_path = absolute ? path : File.join(project_root, path)
  end
end
