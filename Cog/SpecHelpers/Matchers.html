    The target Invocation should create a Cogfile where none existed before

    # File lib/cog/spec_helpers/matchers.rb, line 20
def make(path)
  match_maker do
    message { "to [create|not create] #{path}" }
    before do
      @existed = File.exists? path
    end
    test do
      !@existed && File.exists?(path)
    end
  end
end

    Makes it easy to write RSpec matchers

    Here is how the matcher for #show_help is written using this method match_maker method.

    def show_help
  match_maker do
    message { "to [show|not show] the default help text, got #{lines.first.inspect}" }
    test { (%rhelp.*code gen/ =~ lines[1]) }
  end
end

    Within the match_maker block, self is set to an instance of MatchMaker.

    # File lib/cog/spec_helpers/matchers/match_maker.rb, line 83
def match_maker(&block)
  m = MatchMaker.new
  m.instance_eval &block
  m
end

def output(x)
  match_maker do
    message { "to [write|not write] #{x.join "\n"} to STDOUT"}
    test do
      lines.zip(x).all? {|a, b| a.strip == b.to_s.strip}
    end
  end
end
